Website Overview

VCGamers adalah platform e-commerce gaming yang bertujuan memberdayakan para gamer dengan menyediakan platform seamless untuk mengakses semua kebutuhan gaming mereka — mulai dari top up game hingga berbagai produk digital.

VCGamers — gaming marketplace e-commerce platform overview

Website Rewrite — PWA

VCGamers.com telah ditulis ulang sepenuhnya dalam 6 bulan dengan struktur Progressive Web App (PWA) baru. Sebagai SEO Specialist, saya memainkan peran krusial dalam membentuk strategi SEO untuk memastikan website baru sangat mobile-friendly dan selaras dengan panduan SEO modern Google.

Proses rewrite website ke PWA dengan strategi SEO terintegrasi

Built SEO Strategy

1. Short & Clean URL Structure

Mengoptimalkan URL e-commerce menjadi lebih pendek, lebih sederhana, dan lebih bersih. Hal ini meningkatkan user experience dan pengindeksan search engine, menghilangkan URL yang berantakan dan terlalu kompleks.

Before vs After — URL structure optimization

2. Optimize Heading Structure

Membuat kategori produk (H2) dan semua brand menjadi (H3) dengan hierarki yang jelas. Struktur heading mencakup berbagai kategori produk mulai dari game top up, akun, item dan skin, dan masih banyak lagi.

Implementasi hierarchical heading structure (H1–H4)

3. Content on Footer dengan Accordion

Berkolaborasi dengan tim UI/UX untuk membuat footer yang menampilkan konten SEO menggunakan fitur accordion — memastikan konten SEO tersedia tanpa mengganggu pengalaman pengguna.

Footer dengan fitur accordion untuk konten SEO

4. Differentiate Content — Avoid Duplicate

Membuat strategi dengan membedakan konten setiap halaman kategori produk dari halaman fitur "Belanja Cepat" (Gercep Page). Tujuannya agar search engine lebih mudah melakukan crawling dan indexing, sehingga menghindari duplicate content.

Strategi diferensiasi konten untuk menghindari duplicate content

5. Mobile Responsiveness

Strategi SEO ini dirancang untuk mengoptimalkan performa website dan visibilitas search engine dengan fokus pada pengiriman pengalaman mobile yang seamless. Mencakup responsive design, optimasi page load speed, dan mobile first indexing optimization.

Implementasi mobile responsiveness dan lighthouse scores

SEO Technical

Robots.txt, Sitemap & Canonical

Implementasi robots.txt, XML sitemap, dan canonical tags

JSON-LD / Schema (Structured Data)

Implementasi JSON-LD schema untuk halaman Belanja Cepat (Gercep) dan halaman kategori produk, mencakup Product schema, AggregateRating, dan Review schema untuk meningkatkan rich snippets di SERP.

JSON-LD structured data untuk Gercep page dan Product Category page

Hasil Optimasi

2 bulan setelah meluncurkan website yang ditulis ulang dan membangun strategi SEO, kami segera melihat peningkatan performa yang berpotensi meningkatkan revenue/GMV.

General Performance

GSC General Performance — peningkatan +17.86% clicks, +2.29% impressions, +14.75% CTR dalam satu bulan

Category Page Performance — Top Up

Peningkatan signifikan pada halaman kategori produk prioritas: dari 25.9K clicks (April) menjadi 52.6K clicks (Mei) — peningkatan hampir dua kali lipat (~103.09% dalam satu bulan).

Top Up Category Page — clicks hampir dua kali lipat dalam satu bulan
